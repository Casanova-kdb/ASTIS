# ASTIS

ASTIS is an AI-enhanced study task management system designed to support academic task planning, behaviour tracking, and intelligent task prioritisation.

## Current Status

This project is currently in the planning phase.

## Planned Workflow

The project will follow a lightweight personal Scrum workflow:

- Define requirements and user stories before implementation.
- Break features into small development tasks.
- Track progress through sprint planning and sprint review documents.
- Use clear Git commits to record the project iteration process.

## Documentation

Project documentation will be stored under `docs/`.

- `docs/sprints/`: sprint planning and review records
- `docs/diagrams/`: architecture diagrams and ERD files
- `docs/diagrams/astis-mvp-prototype.pdf`: MVP frontend prototype
- `docs/diagrams/astis-database-erd.png`: database ERD

## Project Structure

```text
backend/    Spring Boot REST API and AI recommendation module
frontend/   Vue 3 user interface
docs/       Requirements, system design, user stories, and sprint records
```

## Planned Tech Stack

- Backend: Java, Spring Boot, Spring Web, Spring Data JPA, Spring Security
- Frontend: Vue 3, Vite, Vue Router, Axios
- Database: MySQL
- API Documentation: Swagger / OpenAPI
- Testing: JUnit, Spring Boot Test, Postman

## Development

Implementation details, setup instructions, and API documentation will be added as the backend and related modules are developed.
